# Cambodia Travel Planner

## Project Info
- **Course:** INFO 251 - Web Development II
- **Professor:** Monyrath Buntoun
- **Team:** CHY Singmey, KEN Sreyleap, KIN Kimcheng, SORN Vitou

## About
A web app that helps users explore Cambodia's 25 provinces, check real-time weather, and plan trips with a personal travel planner.

## Key Features
- Browse and search provinces
- View real-time weather (OpenWeather API)
- Drag & drop trip planner (up to 5 days)
- Save, edit, and delete trips
- User login/signup with localStorage
- Fully responsive design

## Technologies
HTML5, CSS3, JavaScript (ES6+), Fetch API, OpenWeather API, LocalStorage

## API Used
**OpenWeatherMap** - for real-time weather data
